A Fresh Start

After moving to Berlin I decided to redesign my portfolio. The old site was becoming difficult to maintain and most of my projects had already moved elsewhere.

Rather than updating this website, I now keep everything on my GitHub profile.

Most of my older repositories are still public.

These days I keep all my code on GitHub under the same username I've used since college.
